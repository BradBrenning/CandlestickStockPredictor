		Starting up application

run_admin.bat - This launches the application in admin mode.
		Admins have the ability to train and test
		the model. As well as retrieve historical
		data.

run_user.bat - This launches the application in user mode.
	       Users can only see all tradable stocks and
	       see a live view of the stock. Also allowed
	       to mimic trading with a fake portfolio.


		Additional information

bin/ - folder contains the compiled application. This was
       left incase you do not have environmental variables
       setup with JavaFX. You would not be able to compile.

compile.bat - Only use if correctly setup javaFX.

src/ - Contains all raw code for the project to run.

lib/ - Contains library for connecting to SQL (Not used)

data/ - Contains raw files used in application.
	Additional README in folder for further explanation.
